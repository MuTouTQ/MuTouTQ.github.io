Solr 8.6 Started

1. Solr System Requirements

You will need the Java Runtime Environment (JRE) version 1.8 or higher.
你需要使用 JRE JRE 1.8或更高版本。

2.Installing Solr

可以在 Solr 网站上找到 Solr，下载最新版本的 Solr https://lucene.apache.org/Solr/downloads.html。

这里有3个独立的包：

  • solr-8.6.0.tgz for Linux/Unix/OSX systems。适用于 Linux/Unix/OSX 系统的 solr-8.6.0. tgz
  • solr-8.6.0.zip for Microsoft Windows systems。适用于 Windows 系统的 solr-8.6.0.zip
  • solr-8.6.0-src.tgz the package Solr source code. This is useful if you want to develop on Solr without using the official Git repository. Solr-8.6.0-src.tgz 包 Solr 源代码。如果您想在 Solr 上开发而不使用正式的 Git 存储库，这是非常有用的。
